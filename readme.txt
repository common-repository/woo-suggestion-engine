== Description ==

 The easiest way helps you sell more products by search engine and suggested product form your WooCommerce store.

>[Documents]() | [Facebook group](https://www.facebook.com/groups/villatheme "VillaTheme")


### Important Notice:
- Plugin works based on WooCommerce plugin.
- It is released on WordPress.org and you can use plugin as free to build themes for sale.

###FEATURES

&#9658; **Search Engine For WooCommerce:**

- **Fuzzy search**

- **product search:** Allow searching products by some product field with the product's name

- **Result information:** Set the product information you want to show on the results

- **Synonyms:**Allow returning more relevant search results by using synonyms

- **Suggested default:**Allow show suggested products if no result is found

- **Product categories & product tags:**Allow showing product tags/ product categories if found in the results. You can set the title and the number of product categories and product tags to display

&#9658; **Suggested products For WooCommerce:**

- **Create unlimited Shortcode to display the list of suggested products**

- **Hide related products:** Hide related products on the single product pages.

- **Infinite suggestions and set positions to display suggested products on some pages:** this option allows you to put the product suggestion list on some pages. The pages are the Search results page, Shop page, Product category pages, Single product pages, Cart page, and Check out page.

1. Activate the plugin through the 'Plugins' menu in WordPress
